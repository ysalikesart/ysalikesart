Kirby

it's just kirby

This is a code art piece that I made through a JavaScript variant called p5js. I initially pitched this idea because I saw YouTuber SuperMarioLogan's drawings of cartoon characters so I settled with Kirby for this beginning code project. I wanted to make it look like your average CalArts + MSPaint design so I did. I made Sword Kirby from Kirby's Return to Dreamland in CalArts + MSPaint style. The background is inspired by a video tutorial that uses a background similar to what I've made. I'm honestly proud of the art I made, even though it is not my best work, but I love it anyway. Hopefully Cartoon Network hires me in the future if they see this piece. Nah, just kidding lol.
